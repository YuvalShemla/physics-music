# Media Assets to Download

Each station has its own subfolder: drop files into `exhibition/assets/sNN/` with the
**exact filenames** below (e.g. `assets/s03/s03_audio.mp3`).
The exhibition auto-detects them on page load — no code changes needed.

**Done so far:** ✅ s03 complete (physics, music, center images + audio).

## How each station uses its media

| Slot | File | Role | Priority |
|---|---|---|---|
| The Physics (left frame) | `sNN_physics.jpg` | Image of the physics — apparatus, diagram, or physicist | **Required** |
| The Music (right frame) | `sNN_music.jpg` | Image of the musician / performance | **Required** |
| The Connection (sound) | `sNN_audio.mp3` | The work itself — **auto-plays 3 seconds after you arrive at a station** (SPACE pauses/resumes) | **Required** |
| Center frame | `sNN_center.jpg` | Optional connection visual — score page, mapping diagram, artwork | Optional |
| Center frame (video) | `sNN_video.mp4` | Optional — replaces the center image with a video screen (V key) | Optional |

So the core shopping list is **3 files × 10 stations**: two images and one audio excerpt each
(except Station 05/Cage — deliberately silent, images only). The YouTube performance links are
already built into every station's info panel ("Watch Performance ↗") and need no download.

**Formats:** images must be `.jpg`, audio `.mp3`, video `.mp4` (H.264). Rename/convert after downloading.
**Audio tip:** trim each excerpt to the best 60–120 seconds — visitors won't listen longer.
